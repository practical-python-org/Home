What version of Python should I install?

We recommend installing Python 3.10.xx. Installing the LATEST version of python is NOT recommended, as many libraries are not yet supported.

What IDE should I use?

Thonny, VSCode, Pycharm... are all among the most popular ones, however this is a very personal decision. You'll have to try them all! Check the #resources in our discord server for links.

Where do I start learning Python?

There are a number of free and paid resources online that anyone can access. You can check the #resources on our discord server

I'm trying to learn, but it's not working...

I always explain learning code like this...

Imagine you are building a house. You have to do foundations, structure, roofing, plumbing, electrical.... so on and so on. This... is your program. You might learn how a hammer works, but then you can only hammer a nail. And then maybe you learn how a saw works, and then you can use the saw to cut wood and join them with nails. My advice is, learn how the tools in your toolbox work, and interact with each other... and only then you can build a house. knowing how to use a hammer, or a screwdriver, or a saw are... fine... on their own, but when you know how they interact, you can do so much more. These videos you're watching are teaching you the tools you will use for the rest of your career.
